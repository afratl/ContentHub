# ContentHub

A Django-based content and blog management platform where users can explore posts by category, view blog details, interact with likes and comments, and manage content through the admin panel.

## Features

* Blog post listing
* Category-based content organization
* Blog detail pages
* Like functionality
* Image upload support
* User login and registration pages
* Admin panel for content management

## Technologies

* Python
* Django
* HTML
* CSS
* JavaScript
* Bootstrap
* SQLite

## Installation

```bash
git clone https://github.com/afrat1/ContentHub.git

cd ContentHub

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
